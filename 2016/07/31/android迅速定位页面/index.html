android迅速定位页面

Jul 31, 2016

大家是不是经常会遇到别人问自己某个页面是什么Activity,刚开始还能耐心的回答后面慢慢自己也就烦躁了。

今天教大家一个简单快速定位页面是哪个Activity或者fragment的方法。

只适合Debug模式

比如下面这个页面

定位步骤

1. 模拟器或者手机连接电脑并打开要定位的页面

2. 点击Android Monitor左边的System Infomation 中的Actvity Manager State

3. 点击Actvity Manager State 会出现下面的界面 其实就是activity的堆栈信息

当然在栈顶就是当前模拟器或者手机打开的页面及上面那个页面叫做AboutActivity
如果要定位fragment就在当前activity地下有他所包含的fragment
